# Local Development Setup

This course can be completed entirely in your browser using GitHub Codespaces. If you prefer working locally, follow the guide below.

- Full local setup guide (Windows/macOS/Ubuntu):
  - View on GitHub: https://github.com/dipaish/pythonpro26/blob/main/LOCAL_SETUP_GUIDE.md

## Quick Start

1. Install Python 3.11+
2. Install Git
3. Install Visual Studio Code
4. Clone the repository and open it in VS Code

```bash
# Clone
git clone https://github.com/dipaish/pythonpro26.git
cd pythonpro26

# Start with Part 1 tasks
cd part1/part1Exercises/tasks
python grade_part1.py
```

## Prefer zero-setup?

Use GitHub Codespaces (recommended):

1. Fork the repository
2. Click "Code" → "Codespaces" → "Create codespace on main"
3. Start coding. Python and VS Code extensions are preinstalled.

More details: .devcontainer/README.md (in the repository root)
